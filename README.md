# CupHead
An implementation of the game CupHead for PC written in Java.

### Requirements
  - JDK 1.8
  - javafx

### Usage
First you must register a new ID, login, enjoy the game :)<br />
Use `SPACE` to shoot, `TAB` to change the weapon, arrow-keys to move.<br />
